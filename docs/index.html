First-time Setup

Complete these steps once before using the trigger above.

  1. Push this repository to GitHub with the .github/workflows/organize.yml file included.
  2. Run python src/main.py locally once (dry-run is fine) to generate token.json via the browser OAuth flow.
  3. Go to Settings → Secrets and variables → Actions and add two repository secrets:
    • GOOGLE_CREDENTIALS_JSON — paste the entire contents of credentials.json
    • GOOGLE_TOKEN_JSON — paste the entire contents of token.json
  4. Go to Settings → Pages and set the source to Deploy from a branch, branch main, folder /docs. Your site will appear at https://<owner>.github.io/<repo>/.
  5. Create a Personal Access Token with the repo scope and paste it into the form above.
